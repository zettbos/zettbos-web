Tanya Jawab Santai

Admin saya gaptek Excel, emang bisa pake?

Bisa banget. Admin Anda cuma perlu bisa klik tombol dan isi formulir doang. Gak perlu jago rumus. Sistemnya emang kita bikin biar gampang dipake sama orang awam sekalipun. Kita ajarin sampe bisa.

Nanti kalau error gimana?

Selama rumusnya gak diutak-atik (nanti kita kunci biar aman), sistemnya bakal stabil. Kalau pas awal-awal dipake ada error atau bug, wajar kok, langsung kabarin aja biar kita benerin gratis.

Datanya aman gak nih?

Aman pol. Datanya tetep ada di Google Drive atau komputer kantor Anda sendiri. Kita cuma "tukang" yang masuk buat bangun rumahnya, kuncinya tetep Anda yang pegang. Kita gak simpen data Anda di server kami.

Kenapa gak pake software akuntansi yang udah jadi aja?

Kalau pake software jadi, Anda yang harus ngikutin cara kerja softwarenya (yang kadang ribet dan banyak fitur gak kepake). Kalau sama kita, sistemnya yang ngikutin kebiasaan kerja di kantor Anda. Plus, gak ada biaya langganan bulanan selamanya.

Mumpung Data Belum Makin Semrawut.

Makin ditunda, makin numpuk PR admin Anda. Mending diberesin sekarang biar bulan depan tim Anda udah bisa napas lega.
